# Privacy Policy

当サイトは、技術文書の日本語での公開を目的として運営しています。

## 1. 個人情報の取得について
当サイトでは、訪問者から氏名、メールアドレス等の個人情報を直接取得するフォームや会員登録機能等は設けていません。

## 2. Cookie等の利用について
当サイトは GitBook のプラットフォーム上で運営されています。  
GitBook における情報の取り扱いについては、GitBook のプライバシーポリシーをご確認ください。

## 3. 外部リンクについて
当サイトには外部サイトへのリンクが含まれる場合があります。  
リンク先サイトにおける情報の取り扱いについては、各サイトの方針をご確認ください。

## 4. 本ポリシーの変更について
本ポリシーは、必要に応じて予告なく変更することがあります。  
変更後のプライバシーポリシーは、当サイト上に掲載した時点で効力を生じるものとします。

## 5. お問い合わせ
本ポリシーに関するお問い合わせは、GitBookが別途定める連絡先までお願いいたします。

## 6. Disclaimer

掲載内容については正確性の確保に努めていますが、翻訳上の誤り、解釈の相違、更新の遅れその他の理由により、原文と内容が異なる場合があります。  
技術的、法的、金融的またはその他重要な判断を行う場合は、必ず公式ドキュメントその他の一次情報をご確認ください。  
内容に差異または疑義がある場合は、必ず公式ドキュメントを優先してください。
